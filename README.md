# Personal Information Manager

This is a beginner Python project that demonstrates the use of:

- Variables
- User input
- Input validation
- String formatting
- Basic calculations
- Python string methods

## Features

- Stores personal information using variables
- Takes user input for favorite food and favorite color
- Validates user input to avoid empty responses
- Calculates age in months
- Displays formatted output using f-strings
- Uses string methods for better formatting
